# Unreal Engine Studio

## Purpose

Unreal Engine Studio is the program area where students learn real-time 3D creation, interactive world building, lighting, materials, asset workflows, Blueprints, and portfolio-ready environment production.

## Target Quarters

- Quarter 1: Unreal Engine Foundations
- Quarter 3: Unreal Engine Production / Castle Environment

## Q1 Focus

Theme: Unreal Engine Foundations: From First Project to Interactive Mini-World

Students learn how to create a project, navigate the interface, transform objects, manage basic scene setup, use materials and textures, import assets, understand lighting, shape landscapes, place foliage, and preview simple Blueprint-driven interactivity.

## Q3 Focus

Theme: Fantasy Environment Production: Castle World Capstone

Students revisit Unreal through a production lens and build a fantasy/castle environment using planning, landscape blockout, modular assets, Megascans/Fab/Bridge assets, lighting mood, environmental storytelling, walkthroughs, critique, and portfolio submission.

## Expected Student Products

- First Unreal project
- Clean practice scene
- Material and texture examples
- Imported mesh evidence
- Lit environment screenshots
- Landscape and foliage scene
- Interactive mini-world preview
- Fantasy/castle environment capstone
- Cinematic screenshots
- Walkthrough or screen capture
- Portfolio-ready reflection
