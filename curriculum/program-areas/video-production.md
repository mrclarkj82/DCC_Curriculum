# Video Production Studio

## Purpose

Video Production Studio is the area of the website where students access video production projects, production updates, broadcast desk materials, submission instructions, and eventually hosted media evidence.

## Clarifications

- This area is not an in-browser video editor.
- Students will still edit video using external tools selected by the teacher or school.
- The website should eventually collect and host student video evidence, project links, thumbnails, files, reflections, production notes, and teacher feedback.
- The website should eventually include a Broadcast Desk Updates-style section.
- Broadcast Desk Updates should function like teacher-created production posts or project hubs that students can open, follow, and submit work into.

## Possible Future Content Types

- Broadcast Desk Update
- Video Production Project
- B-roll Challenge
- Interview Package
- News Segment
- Studio Role Assignment
- Editing Checkpoint
- Final Video Submission
- Reflection
- Peer Critique
- Portfolio Piece
