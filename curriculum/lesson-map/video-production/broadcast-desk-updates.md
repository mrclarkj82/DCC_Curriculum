# Broadcast Desk Updates

Broadcast Desk Updates are teacher-created production posts that can contain announcements, video references, project requirements, shot lists, assignment links, deadlines, submission instructions, and student upload areas.

They should function like production hubs that students can open, follow, and submit work into.

## Possible Update Types

- Weekly broadcast update
- Production crew update
- Segment assignment
- Studio role assignment
- Deadline reminder
- Reshoot request
- Final export instructions
- Reflection/checkpoint post

## Future Fields

- update ID
- title
- programAreaId
- classId
- publish date
- status
- description
- linked resources
- related project IDs
- submission requirements
- teacher notes
- student-facing instructions
