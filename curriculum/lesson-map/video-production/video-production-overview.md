# Video Production Studio Overview

## Purpose

Video Production Studio gives students a central place to access production projects, teacher updates, submission instructions, resources, deadlines, and eventually hosted media evidence.

## How Students Will Use This Area

Students should eventually open current production work, review teacher instructions, access links or reference videos, submit evidence, write reflections, and track feedback.

## What Teachers Can Post

Teachers should eventually post project briefs, Broadcast Desk Updates, deadline reminders, role assignments, shot lists, linked resources, feedback notes, and portfolio prompts.

## What Students Can Submit

Students should eventually submit video files, hosted media links, thumbnails, scripts, shot lists, production notes, reflections, peer critique responses, and portfolio-ready final exports.

## What Should Eventually Be Hosted

The site should eventually host and organize student video evidence, project links, thumbnails, files, reflections, production notes, teacher feedback, and showcase-ready media items.

## What Should Not Be Built Yet

The site should host and organize video production work, but not edit video. Do not build an in-browser video editor in Phase 1.
