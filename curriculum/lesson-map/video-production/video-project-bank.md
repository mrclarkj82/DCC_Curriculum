# Video Production Project Bank

## VP Project 01: Broadcast Desk Update Response

- Project ID: vp-p01
- Title: Broadcast Desk Update Response
- Skill focus: production communication, media submission, reflection
- Student task: Open an assigned Broadcast Desk Update, follow teacher instructions, and submit the requested evidence.
- Submission evidence: media link or upload, written response, production notes
- Reflection prompt: What did you contribute, what worked, and what would you improve next time?
