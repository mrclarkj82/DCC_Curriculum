# Lesson Maps

This folder contains human-readable curriculum maps that bridge source material and website-ready seed data.

Unreal Engine Studio lesson maps are organized by quarter. Video Production Studio maps, project banks, Broadcast Desk Update models, and production workflows are organized separately so the app can support multiple program areas from the beginning.

Lesson maps should remain readable for teachers while the matching `curriculum/website-data` files provide the structured content foundation for the future website.
