## Folder Notes

- `00_Source Materials/`: original transcripts, teacher notes, examples, rubrics, and reference materials.
- `Unreal Transcript/`: the full Unreal Engine 5 tutorial transcript and related source notes.
- `Video Production Notes/`: teacher notes, Broadcast Desk examples, production routines, equipment workflows, and submission expectations.
- `01_Program Areas/`: high-level planning docs for each major program area.
- `Unreal Engine Studio/`: program overview, quarter plans, and Unreal-specific scope notes.
- `Video Production Studio/`: program overview, project structures, Broadcast Desk planning, and media workflow notes.
- `02_Unreal Q1 Slides/`: generated slide decks for Quarter 1 Unreal lessons.
- `03_Unreal Q1 Assignments/`: student-facing assignment sheets for Quarter 1 Unreal lessons.
- `04_Unreal Q1 Quizzes/`: quizzes and checks for understanding for Quarter 1 Unreal.
- `05_Unreal Q3 Slides/`: generated slide decks for Quarter 3 Unreal production lessons.
- `06_Unreal Q3 Assignments/`: student-facing assignment sheets for Quarter 3 Unreal production lessons.
- `07_Unreal Q3 Quizzes/`: quizzes and checkpoints for Quarter 3 Unreal production.
- `08_Video Production Projects/`: project briefs, rubrics, shot lists, production checklists, and assignment materials.
- `09_Broadcast Desk Updates/`: teacher-created production posts and linked student instructions.
- `10_Student Media Submissions/`: organized student media evidence, links, exports, thumbnails, and reflections.
- `11_Website Lesson and Project Data/`: JSON-ready data, seed files, generated content exports, and schema-aligned content.
- `12_Teacher Dashboard Exports/`: future exports from teacher dashboards, review summaries, and progress reports.
