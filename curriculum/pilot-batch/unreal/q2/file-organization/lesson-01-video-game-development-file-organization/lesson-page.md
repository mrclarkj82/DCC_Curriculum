# Video Game Development File Organization

## Learning Target

I can organize game development files into a professional project folder structure so art, audio, code, levels, documents, exports, plugins, and prefabs are easy to find.

## Why This Matters

Game projects contain many different kinds of files. Models, textures, UI, audio, scripts, shaders, levels, project files, plugins, prefabs, and builds all serve different jobs. A clean folder structure helps a team find files, avoid overwriting work, and prepare a project for development.

## Student Workflow

1. Download and unzip File 2.
2. Create a new main game project folder using the class naming convention.
3. Build the required game development folder structure.
4. Sort project files, art, audio, code, levels, documents, exports, plugins, and prefabs into the correct locations.
5. Confirm that no files remain loose in the main folder.
6. Capture evidence of the finished folder tree.
7. Submit the evidence link and reflection in DCC Creative Studio.
