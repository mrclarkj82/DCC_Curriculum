# Slide Brief: Video Game Development File Organization

Create a 9-slide classroom deck for a 90-minute high school Digital Content Creators block.

## Slide Sequence

1. Title: Video Game Development File Organization
2. Why messy game projects break teamwork
3. The professional game folder structure
4. File type clues: art, audio, code, levels, documents, exports, assets
5. Sorting demo: three example files
6. Tricky cases: UI versus textures, plugins versus scripts, builds versus project files
7. Work time checklist
8. Evidence expectations
9. Exit ticket and cleanup routine

## Visual Direction

Use a game-development studio look with folder-tree diagrams, simple asset icons, and before/after organization. Keep text brief and make categories easy to scan.

## Key Teacher Moves

- Model thinking aloud with file extensions and filename clues.
- Reinforce that no files should remain loose in the main folder.
- Connect organization to Unreal/game project teamwork.

## Student Outcome

Students should leave with a complete organized folder, evidence screenshot, placement map, and reflection.
