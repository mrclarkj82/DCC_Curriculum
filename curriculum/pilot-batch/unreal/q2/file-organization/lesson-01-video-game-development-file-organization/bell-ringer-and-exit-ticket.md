# Bell Ringer and Exit Ticket

## Bell Ringer

What could go wrong in a game project if code, levels, art, audio, plugins, and builds are all mixed together?

## Exit Ticket

Which file type gave you the clearest clue about where it belonged, and which file was most ambiguous?

## Teacher Look-Fors

- Students connect organization to teamwork, broken references, missing assets, or confusing builds.
- Students can name file types that suggest art, audio, code, levels, plugins, prefabs, or exports.
- Students understand that builds are different from project files and source assets.
