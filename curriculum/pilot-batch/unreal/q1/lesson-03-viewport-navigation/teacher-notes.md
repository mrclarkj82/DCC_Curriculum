# Teacher Notes: Viewport Navigation

## Purpose of the Lesson

Students should be able to navigate the Unreal viewport confidently using mouse controls, keyboard movement, focus, view modes, show flags, and game view. This lesson is part of the pilot-ready Unreal Engine Studio content batch.

## Suggested 90-Minute Block Schedule

- 0-8 min: Bell ringer, attendance, and setup check
- 8-18 min: Teacher overview and vocabulary
- 18-35 min: Guided transcript-aligned demo
- 35-65 min: Student work time with teacher circulation
- 65-78 min: Evidence capture, naming, and submission support
- 78-87 min: Extension or cleanup time
- 87-90 min: Exit ticket and save reminder

## Teacher Mini-Lesson Talking Points

- Hold right mouse and use WASD to fly around the scene.
- Use Q and E to move down and up.
- Adjust camera speed with the camera icon and right mouse plus scroll wheel.
- Select an object and press F to focus on it.
- Use Alt plus left mouse to orbit and Alt plus right mouse to zoom around a focused object.
- Switch between Perspective, Top, Lit, Unlit, Wireframe, Show Flags, and Game View.

## Likely Student Issues

- **WASD does nothing:** Check that the right mouse button is being held while pressing WASD.
- **Student gets lost in the scene:** Select an object in the Outliner and press F to focus.
- **Camera moves too fast or too slow:** Adjust Camera Speed or use right mouse plus scroll wheel.
- **Objects seem missing:** Check Show Flags and use defaults if something was hidden.

## Intervention Ideas

- Pair a stuck student with a student who can name the current tool without taking over the mouse.
- Ask students to point to the exact panel or gizmo before explaining the problem.
- Use the transcript timestamps as a rewind point rather than re-teaching the entire process.
- Keep a projected checklist visible during independent work time.

## Extension Ideas

Record a 20-second screen capture showing smooth movement from one object to another and ending in Game View.

## Vocabulary Emphasis

- **Viewport Navigation:** The controls used to move the editor camera around the 3D world.
- **WASD Navigation:** A first-person style movement method used while holding the right mouse button.
- **Camera Speed:** The setting that controls how fast the viewport camera moves.
- **Focus:** The F key action that snaps the viewport to the selected object.
- **View Mode:** A viewport display mode such as Lit, Unlit, or Wireframe.
- **Game View:** A viewport option toggled with G that hides editor-only widgets and shows what the player would see.

## What Evidence To Check Before Students Leave

- Screenshots from at least three different views or camera positions
- One screenshot showing game view or a different view mode
