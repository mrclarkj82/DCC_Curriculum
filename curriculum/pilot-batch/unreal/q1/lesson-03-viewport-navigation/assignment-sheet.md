# Assignment Sheet: Viewport Navigation Scavenger Hunt

## Objective

I can navigate the Unreal viewport confidently using mouse controls, keyboard movement, focus, view modes, show flags, and game view.

## Skill Focus

- viewport navigation
- camera speed
- focus shortcut
- view modes
- game view

## Required Steps

1. Open the Unreal project and use right mouse plus WASD to navigate.
2. Use Q and E to move vertically at least once.
3. Use F to focus on a selected object.
4. Capture screenshots from at least three different views or camera positions.
5. Capture one screenshot showing Game View or a different view mode such as Unlit or Wireframe.

## Naming Convention

Use the class naming convention your teacher assigns. Recommended starter pattern: `LastName_FirstName_UE-Q1-A03`.

## Screenshot/File Evidence Requirements

- Screenshots from at least three different views or camera positions
- One screenshot showing game view or a different view mode

## What To Do If Stuck

- Rewatch the assigned video segment: 00:11:57-00:17:31.
- Check the Common Problems and Fixes section in the lesson page.
- Ask a classmate to name the tool or panel you are trying to use.
- If the project or interface is broken, take a screenshot before asking for help.

## 4-Point Rubric

- **4:** Complete, polished, on time, and clearly meets or extends the stated requirements.
- **3:** Complete and meets the stated requirements.
- **2:** Partially complete or missing required evidence.
- **1:** Attempted but incomplete, unclear, or not functional.
- **0:** Not submitted.

## Extension Challenge

Record a 20-second screen capture showing smooth movement from one object to another and ending in Game View.
