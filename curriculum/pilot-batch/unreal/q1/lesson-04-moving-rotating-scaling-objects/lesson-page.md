## Vocabulary

- **Translate:** Move an object along one or more axes.
- **Rotate:** Turn an object around an axis.
- **Scale:** Change the size or proportions of an object.
- **Gizmo:** The colored handles used to move, rotate, or scale selected objects.
- **Snapping:** A setting that moves, rotates, or scales objects in set increments.
- **Local Transform:** Moving or rotating based on the selected object direction instead of the world direction.

## Common Problems and Fixes

- **Object jumps too far:** Check translation snapping and lower the snap value or turn snapping off.
- **Rotation is hard to align:** Turn rotation snapping on and use predictable increments.
- **Duplicate appears invisible:** Ctrl+D duplicates in the same location; move the copy away to see it.
- **Object floats above or below the floor:** Use careful viewport angles or the End key to snap selected objects down when appropriate.
