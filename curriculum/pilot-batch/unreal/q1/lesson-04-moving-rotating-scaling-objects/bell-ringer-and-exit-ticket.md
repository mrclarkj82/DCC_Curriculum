# Bell Ringer and Exit Ticket: Moving, Rotating, and Scaling Objects

## Bell Ringer Prompt

What is the difference between moving an object and changing the camera view?

## Expected Student Response Type

Short written response, 2-4 sentences or a quick class discussion note.

## Teacher Look-For Notes

- Students connect the skill to digital worlds, media production, games, film, VR, or class workflow.
- Students use everyday language first; technical vocabulary can be introduced during the lesson.
- Responses should reveal prior experience and possible confusion before the demo begins.

## Exit Ticket Prompt

What are the W, E, and R shortcuts used for in Unreal?

## Exit Ticket Grading/Checkoff Guidance

- Check for a clear answer using today's vocabulary.
- Mark complete if the response shows the student understood the main workflow or shortcut.
- Use unclear responses to decide who needs a quick reteach at the start of the next class.
