# Teacher Notes: Moving, Rotating, and Scaling Objects

## Purpose of the Lesson

Students should be able to build a simple starter scene by moving, rotating, scaling, duplicating, and placing objects from the toolbar and Content Drawer. This lesson is part of the pilot-ready Unreal Engine Studio content batch.

## Suggested 90-Minute Block Schedule

- 0-8 min: Bell ringer, attendance, and setup check
- 8-18 min: Teacher overview and vocabulary
- 18-35 min: Guided transcript-aligned demo
- 35-65 min: Student work time with teacher circulation
- 65-78 min: Evidence capture, naming, and submission support
- 78-87 min: Extension or cleanup time
- 87-90 min: Exit ticket and save reminder

## Teacher Mini-Lesson Talking Points

- Select an object and use the move gizmo to translate it.
- Toggle grid snapping and explain Unreal centimeters.
- Use W for move, E for rotate, and R for scale.
- Demonstrate global versus local transform orientation.
- Duplicate with Ctrl+D and with Alt-drag.
- Add objects from the Toolbar and from the Content Drawer.
- Create a simple scene using chairs, a table, a statue, lamps, or available starter assets.

## Likely Student Issues

- **Object jumps too far:** Check translation snapping and lower the snap value or turn snapping off.
- **Rotation is hard to align:** Turn rotation snapping on and use predictable increments.
- **Duplicate appears invisible:** Ctrl+D duplicates in the same location; move the copy away to see it.
- **Object floats above or below the floor:** Use careful viewport angles or the End key to snap selected objects down when appropriate.

## Intervention Ideas

- Pair a stuck student with a student who can name the current tool without taking over the mouse.
- Ask students to point to the exact panel or gizmo before explaining the problem.
- Use the transcript timestamps as a rewind point rather than re-teaching the entire process.
- Keep a projected checklist visible during independent work time.

## Extension Ideas

Use snapping intentionally to align objects and add one note explaining where snapping helped your scene.

## Vocabulary Emphasis

- **Translate:** Move an object along one or more axes.
- **Rotate:** Turn an object around an axis.
- **Scale:** Change the size or proportions of an object.
- **Gizmo:** The colored handles used to move, rotate, or scale selected objects.
- **Snapping:** A setting that moves, rotates, or scales objects in set increments.
- **Local Transform:** Moving or rotating based on the selected object direction instead of the world direction.

## What Evidence To Check Before Students Leave

- Screenshot of completed simple scene
- Short reflection naming at least three tools or shortcuts used
