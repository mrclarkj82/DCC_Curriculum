# Assignment Sheet: Unreal Interface Map

## Objective

I can identify the main Unreal interface panels and explain how each one supports building and organizing a 3D world.

## Skill Focus

- interface awareness
- panel identification
- project organization
- workflow vocabulary

## Required Steps

1. Open the Unreal project from Lesson 01.
2. Capture a clear screenshot of the editor interface.
3. Label the Viewport, Toolbar, Outliner, Details panel, and Content Drawer or Content Browser.
4. Add one short purpose note for each required interface area.
5. Submit the annotated screenshot or teacher-provided form response.

## Naming Convention

Use the class naming convention your teacher assigns. Recommended starter pattern: `LastName_FirstName_UE-Q1-A02`.

## Screenshot/File Evidence Requirements

- Annotated screenshot or form response identifying viewport, toolbar, outliner, details panel, and content drawer

## What To Do If Stuck

- Rewatch the assigned video segment: 00:03:47-00:11:57.
- Check the Common Problems and Fixes section in the lesson page.
- Ask a classmate to name the tool or panel you are trying to use.
- If the project or interface is broken, take a screenshot before asking for help.

## 4-Point Rubric

- **4:** Complete, polished, on time, and clearly meets or extends the stated requirements.
- **3:** Complete and meets the stated requirements.
- **2:** Partially complete or missing required evidence.
- **1:** Attempted but incomplete, unclear, or not functional.
- **0:** Not submitted.

## Extension Challenge

Add two extra interface areas or shortcuts, such as World Settings, F10, Ctrl+Space, or Default Editor Layout.

Which editor panel do you think will help you stay most organized, and why?
