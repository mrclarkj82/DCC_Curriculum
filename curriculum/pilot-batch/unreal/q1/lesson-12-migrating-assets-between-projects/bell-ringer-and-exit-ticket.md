# Migrating Assets Between Projects Bell Ringer and Exit Ticket

## Bell Ringer Prompt

Why might copying only one file fail when moving assets between projects?

## Expected Student Response Type

Short written response in 2-4 sentences. Students should use specific Unreal vocabulary when possible.

## Teacher Look-For Notes

- Students connect the prompt to a visible Unreal workflow or setting.
- Students name what they know and what they still need to test, rather than giving one-word answers.
- Misconceptions from the bell ringer can become quick demo points before independent work.

## Exit Ticket Prompt

Why does Unreal migrate dependencies with an asset?

## Exit Ticket Grading/Checkoff Guidance

- Complete: names a specific change, setting, or skill and explains it clearly.
- Developing: names a general activity but does not connect it to the lesson target.
- Needs follow-up: response is missing, too vague, or shows confusion about the lesson target.
