# Assignment Sheet: First Unreal Project Setup

## Objective

I can create and save a new Unreal Engine project using the correct class setup, project name, and starter content settings.

## Skill Focus

- project creation
- project naming
- Blueprint setting
- starter content
- save location

## Required Steps

1. Open Unreal Engine using the approved class workflow.
2. Create a new Games > Blank project.
3. Leave the project set to Blueprint.
4. Enable Starter Content.
5. Name the project using the class naming convention.
6. Create the project and confirm it opens successfully.

## Naming Convention

Use the class naming convention your teacher assigns. Recommended starter pattern: `LastName_FirstName_UE-Q1-A01`.

## Screenshot/File Evidence Requirements

- Screenshot of project browser or open Unreal project
- Screenshot or written confirmation showing correct project name

## What To Do If Stuck

- Rewatch the assigned video segment: 00:00:00-00:03:47.
- Check the Common Problems and Fixes section in the lesson page.
- Ask a classmate to name the tool or panel you are trying to use.
- If the project or interface is broken, take a screenshot before asking for help.

## 4-Point Rubric

- **4:** Complete, polished, on time, and clearly meets or extends the stated requirements.
- **3:** Complete and meets the stated requirements.
- **2:** Partially complete or missing required evidence.
- **1:** Attempted but incomplete, unclear, or not functional.
- **0:** Not submitted.
