# Teacher Notes: Creating Your First Unreal Project

## Purpose of the Lesson

Students should be able to create and save a new Unreal Engine project using the correct class setup, project name, and starter content settings. This lesson is part of the pilot-ready Unreal Engine Studio content batch.

## Suggested 90-Minute Block Schedule

- 0-8 min: Bell ringer, attendance, and setup check
- 8-18 min: Teacher overview and vocabulary
- 18-35 min: Guided transcript-aligned demo
- 35-65 min: Student work time with teacher circulation
- 65-78 min: Evidence capture, naming, and submission support
- 78-87 min: Extension or cleanup time
- 87-90 min: Exit ticket and save reminder

## Teacher Mini-Lesson Talking Points

- Open Unreal Engine from the Epic Games Launcher or shortcut.
- Use Project Browser to choose Games, then Blank.
- Confirm Blueprint is selected.
- Choose a project location students can find again.
- Name the project using the class convention.
- Enable Starter Content and create the project.

## Likely Student Issues

- **Student cannot find Unreal Engine:** Open Epic Games Launcher, go to Unreal Engine > Library, and confirm the installed engine version.
- **Wrong project type selected:** Return to the Project Browser and select Games > Blank before creating the project.
- **Project saved somewhere unknown:** Check the project location before clicking Create and have students record the folder path.
- **Starter Content missing:** Use Add > Add Feature or Content Pack later, or recreate the project if this is the first setup day.

## Intervention Ideas

- Pair a stuck student with a student who can name the current tool without taking over the mouse.
- Ask students to point to the exact panel or gizmo before explaining the problem.
- Use the transcript timestamps as a rewind point rather than re-teaching the entire process.
- Keep a projected checklist visible during independent work time.

## Extension Ideas

Create a short note showing where your project folder is saved and how you would reopen the .uproject file.

## Vocabulary Emphasis

- **Unreal Engine:** A real-time 3D creation tool used for games, film, visualization, and interactive media.
- **Epic Games Launcher:** The launcher used to install and open Unreal Engine.
- **Project Browser:** The startup window where users open recent projects or create new ones.
- **Blank Project:** A project template that starts with minimal prebuilt content.
- **Starter Content:** Sample assets that can be added to a new project for practice.
- **Blueprint:** Unreal visual scripting used instead of writing C++ for many beginner interactions.

## What Evidence To Check Before Students Leave

- Screenshot of project browser or open Unreal project
- Screenshot or written confirmation showing correct project name
