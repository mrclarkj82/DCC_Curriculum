# Three-Light Mood Study Assignment Sheet

## Objective

I can use different Unreal light types to create a clear mood and visual focus in a scene.

## Skill Focus

- point light setup
- spotlight or rectangle light control
- directional light and skylight awareness
- mood and focal point
- lighting evidence screenshots

## Required Steps

1. Open a small scene with at least one object, wall, prop, or focal area.
2. Create a first lighting setup using a point light.
3. Create a second lighting setup using a spotlight or rectangle light.
4. Create a third lighting setup using a directional light, skylight, or environment lighting.
5. Adjust intensity, color temperature, color, angle, or shadow softness intentionally.
6. Capture one screenshot for each lighting setup.
7. Upload screenshots to Google Drive and paste share links into DCC Creative Studio.
8. Write a reflection explaining which setup created the strongest mood and why.

## Naming Convention

Use this format when naming screenshots or exported evidence: LastName_FirstName_UE-Q1-A14_Description.

## Google Drive Link Evidence Requirements

Upload screenshots, short screen recordings, or written evidence to Google Drive. Paste share links into DCC Creative Studio. Do not upload raw files directly to the website.

- Google Drive link to a screenshot of setup using point light.
- Google Drive link to a screenshot of setup using spotlight or rectangle light.
- Google Drive link to a screenshot of setup using directional light, skylight, or environment lighting.
- Short reflection explaining which setup created the strongest mood and why.

## What To Do If Stuck

- Use point lights for all-direction local light.
- Use spotlights or rectangle lights when you want a directed beam or surface-like source.
- Use directional light for sun-like direction and skylight for environmental fill.
- Change one setting at a time so you can explain what changed.

## 4-Point Rubric

- 4: Complete, polished, on time, and clearly meets or extends the stated requirements.
- 3: Complete and meets the stated requirements.
- 2: Partially complete or missing required evidence.
- 1: Attempted but incomplete, unclear, or not functional.
- 0: Not submitted.

## Extension Challenge

Create a fourth lighting setup inspired by a real place such as a stage, hallway, sunset, studio desk, or game checkpoint.

## Studio Challenge

Use lighting to make one clear focal point without moving the camera or adding text labels.

## Reflection Prompt

Which light type made the biggest difference in mood, and which setting caused that change?
