# Static Mesh Import and Material Setup Assignment Sheet

## Objective

I can import a static mesh and place it correctly in an Unreal scene.

## Skill Focus

- Static meshes are Unreal 3D object assets
- Import options affect how an FBX enters the project
- Folder organization keeps asset sets readable
- Static meshes need materials to look finished
- Default material slots make repeated placements easier

## Required Steps

1. Create a folder for the imported asset set.
2. Import the FBX static mesh into the correct folder.
3. Review import options and reset to default if needed.
4. Choose not to auto-create materials if the class workflow uses a master material.
5. Place the imported mesh in the scene.
6. Import or locate the needed texture maps.
7. Create a material instance from the master material.
8. Assign the correct texture maps to the material instance.
9. Apply the material instance to the mesh.
10. Open the Static Mesh Editor and set the default material slot if appropriate.
11. Capture screenshots of the imported asset and scene result.

## Naming Convention

Use this format when naming screenshots or exported evidence: LastName_FirstName_UE-Q1-A11_Description.

## Screenshot/File Evidence Requirements

- Screenshot of the imported static mesh in the Content Drawer.
- Screenshot of the mesh placed in the scene.
- Screenshot of the material instance or default material slot used on the mesh.
- Short reflection explaining what to check when an imported mesh looks wrong.

## What To Do If Stuck

- Importing assets into a messy or incorrect folder.
- Letting Unreal create unwanted materials when the class workflow uses a master material.
- Forgetting to assign a material to the static mesh.
- Only changing one placed copy instead of setting the default material slot when needed.
- Not checking scale before building a scene around the imported asset.

## 4-Point Rubric

- 4: Complete, polished, on time, and clearly meets or extends the stated requirements.
- 3: Complete and meets the stated requirements.
- 2: Partially complete or missing required evidence.
- 1: Attempted but incomplete, unclear, or not functional.
- 0: Not submitted.

## Extension Challenge

Create a small prop arrangement using at least three duplicates of the imported mesh with different transforms.

## Reflection Prompt

Explain what you should check if an imported mesh is the wrong size, has the wrong material, or appears unfinished.
