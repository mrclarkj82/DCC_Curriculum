# First Custom Material Assignment Sheet

## Objective

I can create a basic Unreal material and control how it looks using base color, roughness, metallic, and simple texture-driven variation.

## Skill Focus

- The Material Editor uses a node graph
- Base Color controls the main visible color
- Roughness controls how shiny or matte the surface feels
- Metallic controls whether the surface behaves like metal
- Apply and Save update the material for the level

## Required Steps

1. Create or open a material example level.
2. Create a new material using naming convention M_.
3. Open the material graph.
4. Add a color/vector value to base color.
5. Add scalar values for roughness and metallic.
6. Press Apply and Save after changes.
7. Apply the material to an object.
8. Optional: add a noise/texture sample to break up roughness or color variation.

## Naming Convention

Use this format when naming screenshots or exported evidence: LastName_FirstName_UE-Q1-A07_Description.

## Screenshot/File Evidence Requirements

- Screenshot of the material graph.
- Screenshot of the material applied to an object.
- Short reflection explaining base color, roughness, and metallic.

## What To Do If Stuck

- Forgetting to press Apply after changing the material graph.
- Using metallic values like 0.5 before understanding the basic PBR workflow.
- Connecting values to the wrong material input.
- Submitting only the final object screenshot without the material graph.

## 4-Point Rubric

- 4: Complete, polished, on time, and clearly meets or extends the stated requirements.
- 3: Complete and meets the stated requirements.
- 2: Partially complete or missing required evidence.
- 1: Attempted but incomplete, unclear, or not functional.
- 0: Not submitted.

## Extension Challenge

Create a comparison board with three objects using the same base color but different roughness values.

## Reflection Prompt

Explain what Base Color, Roughness, and Metallic each changed in your material.
