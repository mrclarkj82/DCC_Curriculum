## Vocabulary

- Material: An Unreal asset that defines how a surface looks when rendered.
- Material graph: The node-based workspace used to build and edit a material.
- Node: A block in the graph that stores or processes information before sending it to a material input.
- Base color: The main visible color of a material before lighting effects are applied.
- Roughness: A value that controls how sharp or blurry reflections appear on a surface.
- Metallic: A value that controls whether a surface behaves like metal or a non-metal.
- PBR: Physically based rendering, a workflow that uses realistic material values such as metallic and roughness.
- Scalar value: A single number value, often used for settings such as Roughness or Metallic.
- Vector/color value: A multi-number value often used to represent RGB color.
- Apply: The material editor action that updates the material so changes appear in the level.

## Assignment Summary

Students create a material, connect a color to base color, adjust roughness and metallic values, apply the material to an object, and submit evidence of both the material graph and the result in the level.

## Submission Checklist

- Screenshot of the material graph.
- Screenshot of the material applied to an object.
- Short reflection explaining base color, roughness, and metallic.

## Extension Challenge

Create a comparison board with three objects using the same base color but different roughness values.

## Exit Ticket

What happens if you change a material but forget to press Apply?

## Common Problems and Fixes

- Forgetting to press Apply after changing the material graph.
- Using metallic values like 0.5 before understanding the basic PBR workflow.
- Connecting values to the wrong material input.
- Submitting only the final object screenshot without the material graph.

## Student-Facing Help

- Use the lesson vocabulary when explaining your work.
- Capture screenshots that clearly show the required skill.
- If something does not update, check the exact setting, connection, or Apply/Save step named in the lesson.
