# Texture Map Material Build Assignment Sheet

## Objective

I can import texture maps and connect base color, roughness, and normal maps correctly to build a more believable material.

## Skill Focus

- Texture assets appear in the Content Drawer
- Texture samples feed image data into the material graph
- Base Color maps carry color information
- Roughness and Normal maps carry non-color data
- Texture Coordinate and Multiply nodes can adjust tiling or scale

## Required Steps

1. Import or locate a base color texture.
2. Import or locate a roughness texture.
3. Import or locate a normal map.
4. Confirm sRGB is on for color textures.
5. Confirm sRGB is off for roughness/masks and normal maps.
6. Confirm normal map compression/settings if applicable.
7. Create a material using M_ naming convention.
8. Connect base color to Base Color.
9. Connect roughness to Roughness.
10. Connect normal map to Normal.
11. Use texture coordinate/multiply setup or equivalent to adjust tiling/scale.
12. Apply and save the material.
13. Apply it to a floor or object.

## Naming Convention

Use this format when naming screenshots or exported evidence: LastName_FirstName_UE-Q1-A08_Description.

## Screenshot/File Evidence Requirements

- Screenshot of imported texture assets.
- Screenshot of material graph connections.
- Screenshot of material applied in the level.
- Short reflection explaining what the normal map adds.

## What To Do If Stuck

- Leaving sRGB on for a roughness map.
- Using the normal map as a color texture instead of connecting it to Normal.
- Forgetting to Apply and Save after wiring the graph.
- Making the texture repeat too often because tiling/scale was not adjusted.
- Jumping ahead to material instances before the base material is understood.

## 4-Point Rubric

- 4: Complete, polished, on time, and clearly meets or extends the stated requirements.
- 3: Complete and meets the stated requirements.
- 2: Partially complete or missing required evidence.
- 1: Attempted but incomplete, unclear, or not functional.
- 0: Not submitted.

## Extension Challenge

Create two versions of the same material with different tiling values and label which one feels more believable for the object scale.

## Reflection Prompt

Explain what the normal map adds to the material and why the non-color map settings matter.
