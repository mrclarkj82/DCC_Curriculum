# Post-Process Exposure Test Assignment Sheet

## Objective

I can use a post-process volume to control exposure and camera-based visual effects in an Unreal scene.

## Skill Focus

- Post Process Volume applies camera and image effects
- Exposure settings control rendered brightness
- Metering Mode can be set to Manual for more control
- Infinite Extent Unbound lets the volume affect the whole level
- Bloom, Vignette, and Saturation change the image style

## Required Steps

1. Create or duplicate a level for post-process testing.
2. Build a simple hut or dark/bright test space.
3. Observe auto exposure behavior.
4. Add or select a post-process volume.
5. Set metering mode to manual.
6. Adjust exposure compensation.
7. Enable infinite extent unbound when appropriate.
8. Experiment with at least one effect such as bloom, vignette, or saturation.

## Naming Convention

Use this format when naming screenshots or exported evidence: LastName_FirstName_UE-Q1-A06_Description.

## Screenshot/File Evidence Requirements

- Screenshot showing auto exposure/problem situation.
- Screenshot showing manual exposure/post-process result.
- Short reflection explaining what changed.

## What To Do If Stuck

- Changing viewport exposure preview settings and expecting the game result to change.
- Forgetting to enable the checkbox beside a post-process property.
- Leaving the camera outside a bounded Post Process Volume.
- Using too many effects at once so the evidence is hard to read.

## 4-Point Rubric

- 4: Complete, polished, on time, and clearly meets or extends the stated requirements.
- 3: Complete and meets the stated requirements.
- 2: Partially complete or missing required evidence.
- 1: Attempted but incomplete, unclear, or not functional.
- 0: Not submitted.

## Extension Challenge

Create two mood versions of the same view, such as neutral exposure and dramatic high-contrast exposure, and label the setting that changed.

## Reflection Prompt

Explain which post-process setting changed the scene most and how your screenshots prove it.
