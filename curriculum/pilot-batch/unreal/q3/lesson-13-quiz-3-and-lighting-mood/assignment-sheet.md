# Lighting Mood Pass Assignment Sheet

## Objective

I can review scene-building concepts and improve lighting mood while documenting how lighting changes the castle environment.

## Skill Focus

- quiz review
- lighting mood
- atmosphere
- screen recording
- production log

## Required Steps

1. Complete or review Q3 Quiz 3 when assigned.
2. Choose a lighting mood for the castle environment.
3. Adjust lighting direction, intensity, color, or time-of-day feel.
4. Capture before/after evidence from the same view.
5. Update the production log with the lighting mood decision.

## File / Project Naming Convention

Use this format when naming project folders, screenshots, recordings, production logs, and written evidence: LastName_FirstName_UE-Q3-A13_Description.

## Evidence Requirements

- Before/after screenshots showing lighting mood.
- Daily documentary clip showing lighting adjustment.
- Production log entry for today.

## Google Drive Link Submission Requirements

- Upload screenshots, screen recordings, camera clips, production log notes, or documentary draft evidence to Google Drive or YouTube according to teacher directions.
- Set link sharing so the teacher can view the evidence.
- Paste the evidence link into DCC Creative Studio.
- Do not upload raw media files directly to the website.
- If the evidence is a folder, make sure every required file inside the folder is viewable.

## What To Do If Stuck

- Re-read the learning target and identify the one visible skill you need to prove today.
- Capture a short before clip before making large changes.
- Ask whether the issue is project organization, camera view, scale, terrain, materials, assets, lighting, or evidence capture.
- Use screenshots plus production notes if screen recording fails.
- Keep working on the castle project while documenting the decision you can explain.

## Four-Point Rubric

- 4: Complete, polished lighting mood pass; daily documentary evidence is clear, organized, and strongly explains process decisions.
- 3: Complete lighting mood pass; daily documentary evidence and production log are present.
- 2: Partially complete lighting mood pass; documentary evidence or production log is incomplete or unclear.
- 1: Attempted but incomplete; little process evidence is visible.
- 0: Not submitted.

## Extension Challenge

Create two lighting looks and label the mood of each.

## Studio Challenge

Add one optional behind-the-scenes clip or caption that could help the final documentary explain today's build decision.

## Reflection Prompt

Which lighting change most affected the emotional tone of the castle?
