# Foliage Placement Pass Assignment Sheet

## Objective

I can place trees and foliage intentionally while documenting how natural details support the castle environment.

## Skill Focus

- foliage placement
- environment detail
- performance awareness
- screen recording
- production log

## Required Steps

1. Add or refine trees, grass, or foliage around the castle environment.
2. Use density and placement to support story, route, or scale.
3. Avoid blocking important views or player movement.
4. Record a clip showing one foliage area before and after improvement.
5. Update the production log with what the foliage adds to the scene.

## File / Project Naming Convention

Use this format when naming project folders, screenshots, recordings, production logs, and written evidence: LastName_FirstName_UE-Q3-A11_Description.

## Evidence Requirements

- Screenshot of foliage placed intentionally.
- Daily documentary clip showing foliage improvement.
- Production log entry for today.

## Google Drive Link Submission Requirements

- Upload screenshots, screen recordings, camera clips, production log notes, or documentary draft evidence to Google Drive or YouTube according to teacher directions.
- Set link sharing so the teacher can view the evidence.
- Paste the evidence link into DCC Creative Studio.
- Do not upload raw media files directly to the website.
- If the evidence is a folder, make sure every required file inside the folder is viewable.

## What To Do If Stuck

- Re-read the learning target and identify the one visible skill you need to prove today.
- Capture a short before clip before making large changes.
- Ask whether the issue is project organization, camera view, scale, terrain, materials, assets, lighting, or evidence capture.
- Use screenshots plus production notes if screen recording fails.
- Keep working on the castle project while documenting the decision you can explain.

## Four-Point Rubric

- 4: Complete, polished foliage placement pass; daily documentary evidence is clear, organized, and strongly explains process decisions.
- 3: Complete foliage placement pass; daily documentary evidence and production log are present.
- 2: Partially complete foliage placement pass; documentary evidence or production log is incomplete or unclear.
- 1: Attempted but incomplete; little process evidence is visible.
- 0: Not submitted.

## Extension Challenge

Create two versions of one area: sparse and dense, then compare the mood.

## Studio Challenge

Add one optional behind-the-scenes clip or caption that could help the final documentary explain today's build decision.

## Reflection Prompt

Where did foliage help the environment, and where could it become distracting?
