# Q3 Production Project Setup Assignment Sheet

## Objective

I can restart my Unreal workflow, set up my Q3 castle project, and begin documenting the production process with camera or screen-recorded evidence.

## Skill Focus

- production setup
- project goals
- screen recording
- camera evidence
- production log

## Required Steps

1. Create or open the assigned Q3 Unreal castle project.
2. Set up a clean project folder and naming convention for Unreal files and documentary evidence.
3. Write a short production goal for the castle environment.
4. Record a short screen capture or camera clip showing the project setup process.
5. Start a production log with the date, what you worked on, what you captured, and what you need next.

## File / Project Naming Convention

Use this format when naming project folders, screenshots, recordings, production logs, and written evidence: LastName_FirstName_UE-Q3-A01_Description.

## Evidence Requirements

- Screenshot of the Q3 project setup.
- Written production goal for the castle environment.
- Daily documentary clip or screen recording showing the setup process.
- Production log entry for today.

## Google Drive Link Submission Requirements

- Upload screenshots, screen recordings, camera clips, production log notes, or documentary draft evidence to Google Drive or YouTube according to teacher directions.
- Set link sharing so the teacher can view the evidence.
- Paste the evidence link into DCC Creative Studio.
- Do not upload raw media files directly to the website.
- If the evidence is a folder, make sure every required file inside the folder is viewable.

## What To Do If Stuck

- Re-read the learning target and identify the one visible skill you need to prove today.
- Capture a short before clip before making large changes.
- Ask whether the issue is project organization, camera view, scale, terrain, materials, assets, lighting, or evidence capture.
- Use screenshots plus production notes if screen recording fails.
- Keep working on the castle project while documenting the decision you can explain.

## Four-Point Rubric

- 4: Complete, polished q3 production project setup; daily documentary evidence is clear, organized, and strongly explains process decisions.
- 3: Complete q3 production project setup; daily documentary evidence and production log are present.
- 2: Partially complete q3 production project setup; documentary evidence or production log is incomplete or unclear.
- 1: Attempted but incomplete; little process evidence is visible.
- 0: Not submitted.

## Extension Challenge

Create a documentary folder structure with subfolders for screen recordings, camera footage, voiceover notes, music, and exports.

## Studio Challenge

Add one optional behind-the-scenes clip or caption that could help the final documentary explain today's build decision.

## Reflection Prompt

What part of today would be useful in a future documentary about how the castle was made?
