# Castle Documentary and Portfolio Submission Assignment Sheet

## Objective

I can prepare final castle evidence and organize documentary footage into a clear making-of story.

## Skill Focus

- final critique
- portfolio evidence
- documentary organization
- reflection
- submission

## Required Steps

1. Review the castle environment from player and cinematic views.
2. Select the strongest screenshots, recordings, and production log entries.
3. Organize documentary evidence into a clear beginning, middle, and end.
4. Prepare the final link-based submission according to teacher directions.
5. Write a final reflection explaining what changed from the first Q3 recording to the final castle.

## File / Project Naming Convention

Use this format when naming project folders, screenshots, recordings, production logs, and written evidence: LastName_FirstName_UE-Q3-A16_Description.

## Evidence Requirements

- Final castle screenshots or walkthrough link.
- Organized documentary evidence folder or edited documentary draft link.
- Final production reflection describing the castle build process.
- Production log summary for the Q3 project.

## Google Drive Link Submission Requirements

- Upload screenshots, screen recordings, camera clips, production log notes, or documentary draft evidence to Google Drive or YouTube according to teacher directions.
- Set link sharing so the teacher can view the evidence.
- Paste the evidence link into DCC Creative Studio.
- Do not upload raw media files directly to the website.
- If the evidence is a folder, make sure every required file inside the folder is viewable.

## What To Do If Stuck

- Re-read the learning target and identify the one visible skill you need to prove today.
- Capture a short before clip before making large changes.
- Ask whether the issue is project organization, camera view, scale, terrain, materials, assets, lighting, or evidence capture.
- Use screenshots plus production notes if screen recording fails.
- Keep working on the castle project while documenting the decision you can explain.

## Four-Point Rubric

- 4: Complete, polished castle documentary and portfolio submission; daily documentary evidence is clear, organized, and strongly explains process decisions.
- 3: Complete castle documentary and portfolio submission; daily documentary evidence and production log are present.
- 2: Partially complete castle documentary and portfolio submission; documentary evidence or production log is incomplete or unclear.
- 1: Attempted but incomplete; little process evidence is visible.
- 0: Not submitted.

## Extension Challenge

Create a 60-90 second making-of documentary edit using the best screen recordings, camera clips, and voiceover notes.

## Studio Challenge

Add one optional behind-the-scenes clip or caption that could help the final documentary explain today's build decision.

## Reflection Prompt

How does your documentary evidence show the castle being made over time?
