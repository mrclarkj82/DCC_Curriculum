# Cinematic Screenshot and Post-Process Pass Assignment Sheet

## Objective

I can use post-process and cinematic framing to capture polished castle visuals for the final documentary.

## Skill Focus

- post-process
- cinematic screenshots
- camera framing
- screen recording
- production log

## Required Steps

1. Set up or refine post-process settings for the castle environment.
2. Choose at least two cinematic camera angles.
3. Capture polished screenshots or short cinematic clips.
4. Record a short screen capture showing how one shot was composed.
5. Update the production log with which shot may appear in the final documentary.

## File / Project Naming Convention

Use this format when naming project folders, screenshots, recordings, production logs, and written evidence: LastName_FirstName_UE-Q3-A14_Description.

## Evidence Requirements

- Two cinematic screenshots of the castle environment.
- Daily documentary clip showing shot setup or post-process adjustment.
- Production log entry for today.

## Google Drive Link Submission Requirements

- Upload screenshots, screen recordings, camera clips, production log notes, or documentary draft evidence to Google Drive or YouTube according to teacher directions.
- Set link sharing so the teacher can view the evidence.
- Paste the evidence link into DCC Creative Studio.
- Do not upload raw media files directly to the website.
- If the evidence is a folder, make sure every required file inside the folder is viewable.

## What To Do If Stuck

- Re-read the learning target and identify the one visible skill you need to prove today.
- Capture a short before clip before making large changes.
- Ask whether the issue is project organization, camera view, scale, terrain, materials, assets, lighting, or evidence capture.
- Use screenshots plus production notes if screen recording fails.
- Keep working on the castle project while documenting the decision you can explain.

## Four-Point Rubric

- 4: Complete, polished cinematic screenshot and post-process pass; daily documentary evidence is clear, organized, and strongly explains process decisions.
- 3: Complete cinematic screenshot and post-process pass; daily documentary evidence and production log are present.
- 2: Partially complete cinematic screenshot and post-process pass; documentary evidence or production log is incomplete or unclear.
- 1: Attempted but incomplete; little process evidence is visible.
- 0: Not submitted.

## Extension Challenge

Create a three-shot sequence: establishing shot, detail shot, and player-view shot.

## Studio Challenge

Add one optional behind-the-scenes clip or caption that could help the final documentary explain today's build decision.

## Reflection Prompt

What makes your strongest shot work?
