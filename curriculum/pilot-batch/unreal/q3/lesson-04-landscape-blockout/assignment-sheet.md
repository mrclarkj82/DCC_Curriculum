# Castle Landscape Blockout Assignment Sheet

## Objective

I can block out terrain forms for a castle environment and document how the terrain supports the route, focal point, and story.

## Skill Focus

- landscape blockout
- terrain form
- focal point
- screen recording
- production log

## Required Steps

1. Create or refine rough terrain forms for the castle, route, and surrounding area.
2. Place temporary markers for the castle entrance, focal point, or path.
3. Test the blockout from editor and player perspectives.
4. Record a short screen capture showing before/after terrain decisions.
5. Update the production log with what terrain choice helped the castle read better.

## File / Project Naming Convention

Use this format when naming project folders, screenshots, recordings, production logs, and written evidence: LastName_FirstName_UE-Q3-A04_Description.

## Evidence Requirements

- Top-down screenshot of the landscape blockout.
- Player-view screenshot showing the route or focal point.
- Daily screen recording showing a terrain/blockout decision.
- Production log entry for today.

## Google Drive Link Submission Requirements

- Upload screenshots, screen recordings, camera clips, production log notes, or documentary draft evidence to Google Drive or YouTube according to teacher directions.
- Set link sharing so the teacher can view the evidence.
- Paste the evidence link into DCC Creative Studio.
- Do not upload raw media files directly to the website.
- If the evidence is a folder, make sure every required file inside the folder is viewable.

## What To Do If Stuck

- Re-read the learning target and identify the one visible skill you need to prove today.
- Capture a short before clip before making large changes.
- Ask whether the issue is project organization, camera view, scale, terrain, materials, assets, lighting, or evidence capture.
- Use screenshots plus production notes if screen recording fails.
- Keep working on the castle project while documenting the decision you can explain.

## Four-Point Rubric

- 4: Complete, polished castle landscape blockout; daily documentary evidence is clear, organized, and strongly explains process decisions.
- 3: Complete castle landscape blockout; daily documentary evidence and production log are present.
- 2: Partially complete castle landscape blockout; documentary evidence or production log is incomplete or unclear.
- 1: Attempted but incomplete; little process evidence is visible.
- 0: Not submitted.

## Extension Challenge

Add one temporary marker showing where the castle entrance should be.

## Studio Challenge

Add one optional behind-the-scenes clip or caption that could help the final documentary explain today's build decision.

## Reflection Prompt

What terrain choice makes the castle feel more intentional?
