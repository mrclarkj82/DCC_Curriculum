# Castle Composition Checkpoint Assignment Sheet

## Objective

I can review mid-unit concepts and improve the castle composition while capturing evidence of visual decision-making.

## Skill Focus

- quiz review
- composition
- focal point
- screen recording
- production log

## Required Steps

1. Complete or review Q3 Quiz 2 when assigned.
2. Choose a camera/player angle to evaluate castle composition.
3. Adjust placement, height, route, or silhouette to improve readability.
4. Capture before/after screenshots or video from the same angle.
5. Update the production log with the composition decision.

## File / Project Naming Convention

Use this format when naming project folders, screenshots, recordings, production logs, and written evidence: LastName_FirstName_UE-Q3-A09_Description.

## Evidence Requirements

- Before/after composition screenshots.
- Daily documentary clip showing a composition decision.
- Production log entry for today.

## Google Drive Link Submission Requirements

- Upload screenshots, screen recordings, camera clips, production log notes, or documentary draft evidence to Google Drive or YouTube according to teacher directions.
- Set link sharing so the teacher can view the evidence.
- Paste the evidence link into DCC Creative Studio.
- Do not upload raw media files directly to the website.
- If the evidence is a folder, make sure every required file inside the folder is viewable.

## What To Do If Stuck

- Re-read the learning target and identify the one visible skill you need to prove today.
- Capture a short before clip before making large changes.
- Ask whether the issue is project organization, camera view, scale, terrain, materials, assets, lighting, or evidence capture.
- Use screenshots plus production notes if screen recording fails.
- Keep working on the castle project while documenting the decision you can explain.

## Four-Point Rubric

- 4: Complete, polished castle composition checkpoint; daily documentary evidence is clear, organized, and strongly explains process decisions.
- 3: Complete castle composition checkpoint; daily documentary evidence and production log are present.
- 2: Partially complete castle composition checkpoint; documentary evidence or production log is incomplete or unclear.
- 1: Attempted but incomplete; little process evidence is visible.
- 0: Not submitted.

## Extension Challenge

Create three labeled camera angles and choose the strongest documentary shot.

## Studio Challenge

Add one optional behind-the-scenes clip or caption that could help the final documentary explain today's build decision.

## Reflection Prompt

What change helped the castle read more clearly?
