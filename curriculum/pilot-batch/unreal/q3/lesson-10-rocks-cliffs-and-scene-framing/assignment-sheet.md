# Rocks, Cliffs, and Framing Pass Assignment Sheet

## Objective

I can use rocks, cliffs, or terrain props to frame the castle environment and document how framing guides the viewer.

## Skill Focus

- scene framing
- rocks and cliffs
- visual guidance
- screen recording
- production log

## Required Steps

1. Add or refine rocks, cliffs, or terrain features around the castle.
2. Use placement to frame the route, entrance, or focal point.
3. Test at least one player-view or camera-view angle.
4. Record a clip showing how framing changed the scene.
5. Update the production log with the framing choice.

## File / Project Naming Convention

Use this format when naming project folders, screenshots, recordings, production logs, and written evidence: LastName_FirstName_UE-Q3-A10_Description.

## Evidence Requirements

- Screenshot showing rocks, cliffs, or terrain framing.
- Daily documentary clip showing a framing decision.
- Production log entry for today.

## Google Drive Link Submission Requirements

- Upload screenshots, screen recordings, camera clips, production log notes, or documentary draft evidence to Google Drive or YouTube according to teacher directions.
- Set link sharing so the teacher can view the evidence.
- Paste the evidence link into DCC Creative Studio.
- Do not upload raw media files directly to the website.
- If the evidence is a folder, make sure every required file inside the folder is viewable.

## What To Do If Stuck

- Re-read the learning target and identify the one visible skill you need to prove today.
- Capture a short before clip before making large changes.
- Ask whether the issue is project organization, camera view, scale, terrain, materials, assets, lighting, or evidence capture.
- Use screenshots plus production notes if screen recording fails.
- Keep working on the castle project while documenting the decision you can explain.

## Four-Point Rubric

- 4: Complete, polished rocks, cliffs, and framing pass; daily documentary evidence is clear, organized, and strongly explains process decisions.
- 3: Complete rocks, cliffs, and framing pass; daily documentary evidence and production log are present.
- 2: Partially complete rocks, cliffs, and framing pass; documentary evidence or production log is incomplete or unclear.
- 1: Attempted but incomplete; little process evidence is visible.
- 0: Not submitted.

## Extension Challenge

Create one cinematic still that uses foreground, middle ground, and background.

## Studio Challenge

Add one optional behind-the-scenes clip or caption that could help the final documentary explain today's build decision.

## Reflection Prompt

How did framing affect where the viewer looks first?
