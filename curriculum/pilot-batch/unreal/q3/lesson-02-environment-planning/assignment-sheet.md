# Castle Environment Plan Assignment Sheet

## Objective

I can plan a fantasy/castle environment and capture planning evidence that explains the intended story and visual direction.

## Skill Focus

- reference gathering
- environment planning
- visual goals
- camera evidence
- production log

## Required Steps

1. Create or update a reference board for the castle environment.
2. Define the castle focal point, route, and mood.
3. Create a simple layout sketch or written scene plan.
4. Record a screen capture or camera clip explaining one planning decision.
5. Update the production log with today's plan, capture notes, and next step.

## File / Project Naming Convention

Use this format when naming project folders, screenshots, recordings, production logs, and written evidence: LastName_FirstName_UE-Q3-A02_Description.

## Evidence Requirements

- Reference board or planning document.
- Simple layout sketch or written scene plan.
- Daily documentary clip explaining one planning decision.
- Production log entry for today.

## Google Drive Link Submission Requirements

- Upload screenshots, screen recordings, camera clips, production log notes, or documentary draft evidence to Google Drive or YouTube according to teacher directions.
- Set link sharing so the teacher can view the evidence.
- Paste the evidence link into DCC Creative Studio.
- Do not upload raw media files directly to the website.
- If the evidence is a folder, make sure every required file inside the folder is viewable.

## What To Do If Stuck

- Re-read the learning target and identify the one visible skill you need to prove today.
- Capture a short before clip before making large changes.
- Ask whether the issue is project organization, camera view, scale, terrain, materials, assets, lighting, or evidence capture.
- Use screenshots plus production notes if screen recording fails.
- Keep working on the castle project while documenting the decision you can explain.

## Four-Point Rubric

- 4: Complete, polished castle environment plan; daily documentary evidence is clear, organized, and strongly explains process decisions.
- 3: Complete castle environment plan; daily documentary evidence and production log are present.
- 2: Partially complete castle environment plan; documentary evidence or production log is incomplete or unclear.
- 1: Attempted but incomplete; little process evidence is visible.
- 0: Not submitted.

## Extension Challenge

Record a short voice memo explaining the intended mood and story of the castle environment.

## Studio Challenge

Add one optional behind-the-scenes clip or caption that could help the final documentary explain today's build decision.

## Reflection Prompt

What planning choice will matter most once you start building?
