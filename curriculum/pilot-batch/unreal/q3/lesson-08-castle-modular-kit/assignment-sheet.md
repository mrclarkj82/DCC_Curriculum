# Castle Modular Kit Assembly Assignment Sheet

## Objective

I can assemble modular castle pieces into a readable structure while documenting construction choices for the final documentary.

## Skill Focus

- modular design
- castle assembly
- scale
- screen recording
- production log

## Required Steps

1. Select or create modular pieces for walls, towers, gates, or platforms.
2. Assemble a readable castle structure or section.
3. Check scale and alignment from editor and player viewpoints.
4. Record a construction clip showing how the castle section changed.
5. Update the production log with the modular pieces used and what still needs work.

## File / Project Naming Convention

Use this format when naming project folders, screenshots, recordings, production logs, and written evidence: LastName_FirstName_UE-Q3-A08_Description.

## Evidence Requirements

- Screenshot of the castle modular section.
- Daily documentary clip showing castle construction progress.
- Production log entry for today.

## Google Drive Link Submission Requirements

- Upload screenshots, screen recordings, camera clips, production log notes, or documentary draft evidence to Google Drive or YouTube according to teacher directions.
- Set link sharing so the teacher can view the evidence.
- Paste the evidence link into DCC Creative Studio.
- Do not upload raw media files directly to the website.
- If the evidence is a folder, make sure every required file inside the folder is viewable.

## What To Do If Stuck

- Re-read the learning target and identify the one visible skill you need to prove today.
- Capture a short before clip before making large changes.
- Ask whether the issue is project organization, camera view, scale, terrain, materials, assets, lighting, or evidence capture.
- Use screenshots plus production notes if screen recording fails.
- Keep working on the castle project while documenting the decision you can explain.

## Four-Point Rubric

- 4: Complete, polished castle modular kit assembly; daily documentary evidence is clear, organized, and strongly explains process decisions.
- 3: Complete castle modular kit assembly; daily documentary evidence and production log are present.
- 2: Partially complete castle modular kit assembly; documentary evidence or production log is incomplete or unclear.
- 1: Attempted but incomplete; little process evidence is visible.
- 0: Not submitted.

## Extension Challenge

Create a labeled modular kit preview showing the pieces used in your castle.

## Studio Challenge

Add one optional behind-the-scenes clip or caption that could help the final documentary explain today's build decision.

## Reflection Prompt

Which modular piece had the biggest effect on the castle silhouette?
