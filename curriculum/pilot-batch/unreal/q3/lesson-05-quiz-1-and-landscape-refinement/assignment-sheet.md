# Landscape Refinement Pass Assignment Sheet

## Objective

I can review early Q3 production expectations and refine the castle landscape while documenting visible progress.

## Skill Focus

- quiz review
- landscape refinement
- before-after evidence
- screen recording
- production log

## Required Steps

1. Complete or review Q3 Quiz 1 when assigned.
2. Choose one landscape problem to improve.
3. Refine terrain shape, scale, or route readability.
4. Capture before/after screenshots or a short screen recording of the refinement.
5. Update the production log with the improvement and remaining issue.

## File / Project Naming Convention

Use this format when naming project folders, screenshots, recordings, production logs, and written evidence: LastName_FirstName_UE-Q3-A05_Description.

## Evidence Requirements

- Screenshot or clip showing the refined landscape.
- Before/after evidence of one improvement.
- Daily documentary clip or screen recording showing refinement work.
- Production log entry for today.

## Google Drive Link Submission Requirements

- Upload screenshots, screen recordings, camera clips, production log notes, or documentary draft evidence to Google Drive or YouTube according to teacher directions.
- Set link sharing so the teacher can view the evidence.
- Paste the evidence link into DCC Creative Studio.
- Do not upload raw media files directly to the website.
- If the evidence is a folder, make sure every required file inside the folder is viewable.

## What To Do If Stuck

- Re-read the learning target and identify the one visible skill you need to prove today.
- Capture a short before clip before making large changes.
- Ask whether the issue is project organization, camera view, scale, terrain, materials, assets, lighting, or evidence capture.
- Use screenshots plus production notes if screen recording fails.
- Keep working on the castle project while documenting the decision you can explain.

## Four-Point Rubric

- 4: Complete, polished landscape refinement pass; daily documentary evidence is clear, organized, and strongly explains process decisions.
- 3: Complete landscape refinement pass; daily documentary evidence and production log are present.
- 2: Partially complete landscape refinement pass; documentary evidence or production log is incomplete or unclear.
- 1: Attempted but incomplete; little process evidence is visible.
- 0: Not submitted.

## Extension Challenge

Add temporary labels or markers that explain the landscape improvement.

## Studio Challenge

Add one optional behind-the-scenes clip or caption that could help the final documentary explain today's build decision.

## Reflection Prompt

Which refinement made the biggest difference, and why?
