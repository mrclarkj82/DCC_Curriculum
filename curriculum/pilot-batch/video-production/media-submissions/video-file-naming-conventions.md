# Video File Naming Conventions

## Recommended Naming Pattern

`LastName_FirstName_ProjectID_Version`

## Examples

- `Garcia_Ana_VP-P02_v1`
- `Garcia_Ana_VP-P02_Final`
- `Garcia_Ana_BRollSequence_v2`

## Unacceptable File Names

- `final.mov`
- `my video.mp4`
- `newnewfinalFINAL.mp4`
- `untitled project.mp4`
- `video for class.mp4`

## Export and Version Examples

- First export: `LastName_FirstName_VP-P02_v1.mp4`
- Revised export: `LastName_FirstName_VP-P02_v2.mp4`
- Final export: `LastName_FirstName_VP-P02_Final.mp4`
- Thumbnail: `LastName_FirstName_VP-P02_Thumbnail.jpg`

## Notes

Use the project ID when possible so teacher dashboards and future imports can connect submissions to the correct project.
