## Questions

### 1. What is a title used for in an edit?

- Readable text such as a title card, name label, or lower third.
- A raw camera file.
- A no-school calendar note.
- A timeline backup folder.

- Correct answer: Readable text such as a title card, name label, or lower third.
- Explanation: Titles communicate text information inside the video.
- Lesson: vp-q2-l06
- Tag: titles

### 2. Which panel is commonly used to adjust title text, position, and selected effect settings?

- Inspector
- Project Manager
- Render Queue
- Media Storage

- Correct answer: Inspector
- Explanation: The Inspector exposes controls for selected clips, titles, transitions, and effects.
- Lesson: vp-q2-l06
- Tag: inspector

### 3. What is a keyframe?

- A point in time that stores a value so a property can change over time.
- The final exported file.
- A folder inside the Media Pool.
- The first day of the grading period.

- Correct answer: A point in time that stores a value so a property can change over time.
- Explanation: Keyframes make motion or property changes happen over time.
- Lesson: vp-q2-l06
- Tag: keyframes

### 4. When should a transition be used?

- When it supports the edit and does not distract from the story.
- Between every single clip no matter what.
- Only before importing footage.
- Only after the final file is exported.

- Correct answer: When it supports the edit and does not distract from the story.
- Explanation: Transitions should serve clarity, pacing, or tone instead of becoming decoration.
- Lesson: vp-q2-l06
- Tag: transitions

### 5. What makes a lower third or title card classroom-ready?

- It is readable, spelled correctly, timed well, and supports the viewer.
- It is as small as possible.
- It covers the entire edit for the whole video.
- It replaces the need to organize clips.

- Correct answer: It is readable, spelled correctly, timed well, and supports the viewer.
- Explanation: Student titles should improve communication without hiding the video.
- Lesson: vp-q2-l06
- Tag: title-readability

### 6. Which page is used to render or export the finished timeline?

- Deliver
- Media
- Edit
- Project Manager

- Explanation: The Deliver page is used to render the timeline into a finished video file.
- Lesson: vp-q2-l07
- Tag: deliver

### 7. What is the render queue?

- A list of export jobs waiting to render.
- A list of raw clips not imported.
- A folder of quiz answers.
- A set of A/B class periods.

- Correct answer: A list of export jobs waiting to render.
- Explanation: Resolve adds export jobs to the render queue before rendering.
- Lesson: vp-q2-l07

### 8. Why should the file name and save location be checked before rendering?

- So the exported file can be found and submitted correctly.
- So the project deletes unused clips.
- So the timeline automatically rewrites itself.
- So the grading period changes.

- Correct answer: So the exported file can be found and submitted correctly.
- Explanation: Naming and location checks prevent lost exports and confusing submissions.
- Lesson: vp-q2-l07
- Tag: file-management

### 9. What is a codec?

- A method used to encode and compress video or audio data.
- A title animation preset only.
- A class join code.
- A type of bin folder.

- Correct answer: A method used to encode and compress video or audio data.
- Explanation: Codecs such as H.264 and H.265 affect how a video file is encoded.
- Lesson: vp-q2-l07
- Tag: codec

### 10. Which export setting is commonly used for a web-friendly class submission?

- H.264 in an MP4 container.
- A missing file path.
- A project file with no rendered video.
- A timeline with no title or final check.

- Correct answer: H.264 in an MP4 container.
- Explanation: H.264 MP4 files are widely playable and practical for class review.

### 11. Why should an editor watch the final timeline before exporting?

- To skip the render queue.
- To delete the project manager.

- Correct answer: To catch gaps, unreadable titles, timing problems, or other mistakes.
- Explanation: A final playback check catches issues before a file is submitted.

### 12. What should a simple motion effect do in a school edit?
