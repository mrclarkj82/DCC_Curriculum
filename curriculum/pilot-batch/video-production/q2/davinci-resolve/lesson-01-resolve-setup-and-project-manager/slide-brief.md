# Slide Brief: DaVinci Resolve Setup and Project Manager

Create a 9-slide classroom deck for a 90-minute high school Digital Content Creators block.

## Slide Sequence

1. Title, learning target, and today's evidence goal.
2. Hook: What is one problem that can happen if an editor starts cutting video before naming, saving, and organizing the project?
3. Why this matters for video production students.
4. Vocabulary mini-map: DaVinci Resolve, Project Manager, Project Library, Resolve page, Post-production.
5. Interface focus: where this workflow happens in DaVinci Resolve.
6. Teacher demo steps: Open DaVinci Resolve and locate the Project Manager. / Create a new project using the class naming convention. / Identify the Media, Cut, Edit, Fusion, Color, Fairlight, and Deliver pages..
7. Common mistakes and fixes.
8. Independent work checklist and evidence expectations.
9. Reflection and exit ticket: Which Resolve page or startup area did you use today, and what job does it help an editor do?

## Visual Suggestions

- Use dark synthwave backgrounds with bright cyan, magenta, and orange labels.
- Include Resolve interface screenshots or placeholders with clear arrows.
- Use short labels and large readable UI callouts instead of paragraphs.
- Show before/after or messy/organized comparisons where possible.

## Speaker Notes

Keep the deck focused on one practical workflow. Students should spend most of the period working in Resolve, not copying notes.
