# Video Production File Organization

## Learning Target

I can organize video production files into a professional project folder structure so footage, audio, graphics, documents, assets, and exports are easy to find.

## Why This Matters

Editors lose time when project files, footage, audio, graphics, documents, and exports all live in one messy folder. Before students begin serious DaVinci Resolve work, they need a repeatable folder structure that keeps source material separate from working files and final exports.

## Student Workflow

1. Download and unzip File 1.
2. Create a main project folder using the class naming convention.
3. Build the required video production folder structure.
4. Sort footage, audio, graphics, documents, assets, project files, and exports into the correct locations.
5. Confirm that no files remain loose in the main folder.
6. Capture evidence of the finished folder tree.
7. Submit the evidence link and reflection in DCC Creative Studio.

## Folder Structure
