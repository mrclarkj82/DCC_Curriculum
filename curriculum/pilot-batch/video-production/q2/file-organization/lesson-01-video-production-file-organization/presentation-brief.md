# Presentation Brief: Video Production File Organization

This premium brief is for creating a 15-20 slide teacher-led PowerPoint for the first Q2 video production organization lesson.

## Audience

High school Digital Content Creators students beginning a Q2 video production workflow. Students may recognize common media files, but many have not practiced organizing a professional editing project folder.

## Lesson Purpose

Students will organize a messy video production ZIP into a folder structure that separates source footage, audio, graphics, documents, project files, reusable assets, and exports. This prepares them for DaVinci Resolve work by building good habits before editing begins.

## Required Resources

- File 1 ZIP: https://drive.google.com/file/d/1lawbe-dxszgGbAr6d5zx4nFPOuU6j4Sk/view?usp=sharing
- File 1 instructions: https://docs.google.com/document/d/1fAgKFFECPnBWXnh4G7CgksESe9JTRV4OOTtsUKeDPjw/edit?usp=sharing

## Essential Understanding

A professional editor can recover and continue work faster when the project folder clearly separates working files, source media, graphics, audio, documents, assets, and exports.

## Suggested 15-20 Slide Structure

1. Title and objective
2. The messy-folder problem
3. Real production consequences: missing media, wrong exports, wasted time
4. Project folder overview
5. `01_Project_Files`
6. `02_Footage`
7. `03_Audio` with Voiceover, Music, and SoundFX
8. `04_Graphics` with Logos, Lower Thirds, Titles, and Images
9. `05_Exports` with Drafts and Final
10. `06_Documents`
11. `07_Assets` with Stock Footage and Templates
12. Filename clues and extension clues
13. Teacher sorting demo
14. Student workflow checklist
15. Evidence expectations
16. Common mistakes to avoid
17. Extension challenge
18. Exit ticket
19. Cleanup and submit
20. Optional reflection discussion

## Slide-By-Slide Teaching Purpose

- Slides 1-3 should establish why organization matters before editing.
- Slides 4-11 should make the folder structure visible and concrete.
- Slides 12-13 should model how to reason from filenames and extensions.
- Slides 14-16 should support independent work and reduce common mistakes.
- Slides 17-20 should close with evidence, reflection, and cleanup.

## Must Include

- Clear folder-tree visual.
- At least six file examples from the ZIP.
- A before/after comparison.
- Evidence checklist.
- No raw upload directions; students submit Drive or Docs links through DCC Creative Studio.

## Tone

Practical, production-focused, and teacher-led. This should feel like a real studio workflow, not a generic computer file lesson.
