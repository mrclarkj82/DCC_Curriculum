# Video Production Folder Organization Assignment Sheet

## Objective

I can organize video production files into a professional project folder structure so footage, audio, graphics, documents, assets, and exports are easy to find.

## Assignment Resources

- File 1 ZIP: https://drive.google.com/file/d/1lawbe-dxszgGbAr6d5zx4nFPOuU6j4Sk/view?usp=sharing
- File 1 instructions: https://docs.google.com/document/d/1fAgKFFECPnBWXnh4G7CgksESe9JTRV4OOTtsUKeDPjw/edit?usp=sharing

## Skill Focus

- project folder structure
- media organization
- file type recognition
- production workflow
- evidence documentation

## Required Steps

1. Open the File 1 instructions.
2. Download File 1 and unzip it on your computer.
3. Create a new main project folder using the class naming convention.
4. Build the required folder structure exactly.
5. Sort project files, footage, audio, graphics, exports, documents, and assets into the correct folders.
6. Use file names and file extensions as clues when sorting.
7. Check that no files are left loose in the main project folder.
8. Capture a screenshot of the completed folder tree.
9. Create a short placement map listing at least twelve files and where you placed them.
10. Upload evidence to Google Drive or Google Docs and submit the share links in DCC Creative Studio.

## Sorting Hints

- `.drp`, `.prproj`, and `.fcpxml` files belong in `01_Project_Files`.
- `.MP4` and `.MOV` source clips belong in `02_Footage`.
- `A###` and `B###` clips are camera footage.
- `PhoneClip` clips should be grouped as phone footage.
- `.wav` and `.mp3` files belong in `03_Audio`.
- `VO_` files belong in `Voiceover`.
- `BackgroundTrack` files belong in `Music`.
- Sound-effect names such as `DoorSlam` and `Footsteps` belong in `SoundFX`.
- `.png`, `.jpg`, `.psd`, and `.aep` files belong in `04_Graphics` or `07_Assets`.
- Draft exports belong in `05_Exports/Drafts`.
- Final exports belong in `05_Exports/Final`.
- `.docx`, `.pdf`, and `.xlsx` files belong in `06_Documents`.

## Naming Convention

Use this format when naming screenshots, written evidence, or submitted files: `LastName_FirstName_VP-Q2-FILEORG_Description`.

## Google Drive Link Evidence Requirements

- Google Drive link to a screenshot showing the completed video production folder tree.
- Google Drive or Docs link listing where at least twelve sample files were placed and why.
- Short reflection explaining two organization decisions that would help an editor work faster.

## What To Do If Stuck

- Look at the file extension first.
- Look for words in the file name such as `VO`, `BackgroundTrack`, `Logo`, `Draft`, `Final`, or `Template`.
- Ask whether the file is source media, a working project file, a graphic, a document, an export, or a reusable asset.

## 4-Point Rubric

- 4: Complete folder structure, all files sorted accurately, evidence is clear, and reflection explains thoughtful workflow decisions.
- 3: Complete folder structure, most files sorted accurately, and all required evidence is submitted.
- 2: Folder structure or sorting is partially complete, or required evidence is missing detail.
- 1: Attempted but incomplete, hard to verify, or many files remain misplaced.
- 0: Not submitted.
