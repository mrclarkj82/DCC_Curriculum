# Slide Brief: Video Production File Organization

Create a 9-slide classroom deck for a 90-minute high school Digital Content Creators block.

## Slide Sequence

1. Title: Video Production File Organization
2. Why messy folders break video projects
3. The professional folder structure
4. File type clues: project files, footage, audio, graphics, documents, exports
5. Sorting demo: three example files
6. Tricky cases: draft versus final, source footage versus export, template versus graphic
7. Work time checklist
8. Evidence expectations
9. Exit ticket and cleanup routine

## Visual Direction

Use a clean production-studio look with screenshots or folder-tree diagrams. Avoid heavy text. Use color coding for folder categories and show before/after organization.

## Key Teacher Moves

- Model thinking aloud with file extensions and filename clues.
- Reinforce that no files should remain loose in the main folder.
- Connect organization to future DaVinci Resolve media management.

## Student Outcome

Students should leave with a complete organized folder, evidence screenshot, placement map, and reflection.
