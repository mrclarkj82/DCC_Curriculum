# Bell Ringer and Exit Ticket

## Bell Ringer

What is one problem that can happen when a video editor keeps footage, audio, graphics, and exports all in one messy folder?

## Exit Ticket

Which file category was easiest to identify today, and which category required the most careful thinking?

## Teacher Look-Fors

- Students connect organization to lost time, broken links, duplicate exports, or missing media.
- Students can name at least one file extension or naming clue that helped them sort accurately.
- Students understand that exports are different from source footage and working project files.
