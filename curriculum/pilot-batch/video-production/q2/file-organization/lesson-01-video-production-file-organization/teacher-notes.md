# Teacher Notes: Video Production File Organization

## Prep

- Confirm students can open the File 1 ZIP and instruction document.
- Decide where students should create the practice project folder: local desktop, OneDrive, Google Drive sync folder, or a class workspace.
- Remind students that the website collects Drive or Docs links only. They should not upload raw folders to DCC Creative Studio.

## Lesson Flow

1. Open with a messy-folder scenario and ask students what could go wrong.
2. Show the folder structure and explain why each top-level folder exists.
3. Model sorting three files out loud: one obvious file, one tricky file, and one export.
4. Release students to build their folders and sort the full ZIP.
5. Circulate for folder naming, no loose files, and evidence quality.
6. Close with a reflection about which organization choices would help an editor work faster.

## Common Misconceptions

- Students may put final exports in the same place as source footage.
- Students may treat After Effects templates as finished graphics instead of reusable templates.
- Students may ignore camera prefixes such as `A001` and `B001`.
- Students may leave files loose in the main folder after building subfolders.

## Evidence Check

Students should submit:

- Screenshot of the finished folder tree.
- Placement map for at least twelve files.
- Reflection naming two workflow decisions.

## Extension

Ask early finishers to write a short handoff note for another editor who has never seen the project.
