# Submission Requirements: B-Roll Sequence

## Acceptable File/Link Types

- video
- link
- fileUpload
- textReflection

## Required Evidence

- Exported B-roll sequence or hosted link
- Shot list or production notes
- Short reflection

## File or Link Expectations

- Use the project naming convention.
- Make hosted links viewable by the teacher.
- Do not include private student media beyond what the teacher requested.
- Do not submit real unpublished school media to public locations without teacher approval.

## Teacher Review Criteria

- Evidence matches the project instructions.
- Media or links open correctly.
- Reflection answers the prompt.
- Production notes are clear enough for teacher feedback.

## Website Boundary

DCC Creative Studio should eventually host, collect, organize, and review this work. It should not edit the video in-browser.
