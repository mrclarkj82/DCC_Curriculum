# Broadcast Desk Update Response

- Project ID: vp-p01
- Program Area ID: video-production
- Program Area: Video Production Studio
- Project Type: broadcast-update-response
- Status: pilot-ready, needs teacher review

## Student-Facing Project Overview

Students open a teacher-created Broadcast Desk Update, follow the instructions, and submit required production evidence or response.

## Production Goal

Read a production update carefully, identify what evidence is required, and submit a complete response using the teacher instructions.

## Required Steps

1. Open the assigned Broadcast Desk Update.
2. Read the student-facing instructions and checklist.
3. Review linked resource placeholders or teacher-provided examples.
4. Complete the requested production response or media evidence.
5. Submit the required file, link, notes, and reflection.

## Equipment/Software Notes

- Teacher-selected camera, phone, school camera, or existing media source as appropriate
- External editing tool selected by the teacher or school if editing is required
- DCC Creative Studio will host and organize submissions later; it will not edit video

## Shot List or Production Checklist

- Confirm the update ID and related project ID.
- Submit the required media file, hosted link, or text response.
- Include production notes if requested.
- Complete the reflection prompt.

## Submission Requirements

- Submitted media link or uploaded file
- Short reflection
- Any required production notes

## Acceptable File/Link Types

- video
- link
- textReflection
- fileUpload

## File Naming Expectations

Use `LastName_FirstName_ProjectID_Version`, such as `LastName_FirstName_VP-P01_v1` or `LastName_FirstName_VP-P01_Final`.

## 4-Point Rubric

- **4:** Complete, polished, on time, and clearly meets or extends the stated requirements.
- **3:** Complete and meets the stated requirements.
- **2:** Partially complete or missing required evidence.
- **1:** Attempted but incomplete, unclear, or not functional.
- **0:** Not submitted.

## Reflection Prompt

What did you contribute, what worked, and what would you improve next time?

## Common Problems

- Students skip the teacher instructions before submitting.
- Links are shared privately and cannot be opened by the teacher.
- The reflection describes the task but not the student contribution.

## Teacher Notes

Use this as a flexible response shell for any teacher-created production update. Replace placeholders with current class instructions before publishing.

## Important Boundary

The website hosts, collects, organizes, and reviews video production work. The website does not edit video.
