# Submission Requirements: Broadcast Desk Update 001: Production Check-In

## Required Submission Evidence

- Media file, link, or production response
- Short reflection or production notes

## Acceptable Formats

- Video file placeholder
- Hosted video link placeholder
- File upload placeholder
- Text reflection or production notes

## Teacher Review Status Definitions

- Draft: work has not been submitted yet.
- Submitted: student has turned in required evidence.
- Reviewed: teacher has checked the evidence.
- Returned: teacher has requested a revision.

## Privacy Reminder

Do not submit private media, real student data, or public links unless your teacher has approved the sharing settings.
