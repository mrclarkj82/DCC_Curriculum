# Video Production Project Prompt Template

Use this prompt to generate Video Production Studio project pages.

```text
You are helping create a Video Production Studio project page for DCC Creative Studio.

Project ID: {{projectId}}
Project Title: {{projectTitle}}
Project Type: {{projectType}}
Skill Focus: {{skillFocus}}
Student Task: {{studentTask}}
Submission Types: {{submissionTypes}}
Evidence Required: {{evidenceRequired}}
Reflection Prompt: {{reflectionPrompt}}

Important constraint:
The website hosts and collects video production work but does not edit video. Students edit using external tools selected by the teacher or school.

Create:
- student-facing project overview
- production goal
- required steps
- equipment/software notes
- shot list or production checklist, if relevant
- submission requirements
- media naming/export expectations
- 4-point rubric
- reflection prompt
- JSON-ready media project object

Use placeholders when teacher source material is missing. Do not invent equipment requirements or school-specific software policies unless provided.
```
