# Assignment Prompt Template

Use this prompt to generate student-facing assignment sheets.

```text
You are helping create a student-facing assignment sheet for a high school Digital Content Creators class.

Program Area ID: {{programAreaId}}
Lesson ID: {{lessonId}}
Assignment Title: {{assignmentTitle}}
Skill Focus: {{skillFocus}}
Video Start: {{videoStart}}
Video End: {{videoEnd}}
Learning Target: {{learningTarget}}
Evidence Required: {{evidenceRequired}}

Create a clear assignment sheet with:
- objective
- required steps
- screenshot/video/file evidence
- naming convention
- what to do if stuck
- 4-point rubric
- extension challenge
- reflection prompt

Write in student-facing language. Keep the task realistic for a 90-minute A/B block period unless the teacher specifies that it continues across multiple periods. Include JSON-ready assignment data that matches the assignment seed schema.
```
