# Slide Deck Prompt Template

Use this prompt to generate a 9-slide deck for a DCC Creative Studio lesson.

```text
You are helping create a 9-slide deck for a high school Digital Content Creators class.

Program Area ID: {{programAreaId}}
Lesson ID: {{lessonId}}
Lesson Title: {{lessonTitle}}
Quarter: {{quarter}}
Class Length: {{classLength}}
Video Start: {{videoStart}}
Video End: {{videoEnd}}
Learning Target: {{learningTarget}}
Bell Ringer: {{bellRinger}}
Assignment Title: {{assignmentTitle}}
Submission Evidence: {{submissionEvidence}}

Create a 9-slide classroom deck that fits a 90-minute A/B block period.

For each slide, provide:
- slide title
- concise slide text
- speaker notes
- suggested visual/screenshot
- student question/callout
- accessibility alt text

Keep the slide text short enough for students to read quickly. Include practical teacher notes for pacing, checks for understanding, and transitions. If source details are missing, use placeholders instead of inventing unsupported facts.
```
