```text
You are helping define media submission requirements for DCC Creative Studio.

Program Area ID: {{programAreaId}}

Submission Type: {{submissionType}}
File Types: {{fileTypes}}
Max File Size: {{maxFileSize}}
Naming Convention: {{namingConvention}}
Reflection Prompt: {{reflectionPrompt}}

Create:
- acceptable file/link types
- naming convention
- evidence checklist
- reflection requirement
- teacher review criteria
- common submission problems
- JSON-ready media submission schema example

The website should eventually collect, organize, host, and review media submissions. It should not edit video in-browser unless a later phase explicitly requests that feature.
```
