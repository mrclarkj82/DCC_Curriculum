# Broadcast Update Prompt Template

Use this prompt to generate Broadcast Desk Updates.

```text
You are helping create a Broadcast Desk Update for Video Production Studio in DCC Creative Studio.

Update ID: {{updateId}}
Title: {{title}}
Class Period: {{classPeriod}}
Publish Date: {{publishDate}}
Due Date: {{dueDate}}
Summary: {{summary}}
Student Instructions: {{studentInstructions}}
Linked Resources: {{linkedResources}}
Related Project IDs: {{relatedProjectIds}}
Submission Requirements: {{submissionRequirements}}

Create:
- teacher-facing update draft
- student-facing update copy
- production checklist
- linked resource list
- submission instructions
- teacher notes
- JSON-ready broadcast update object

Broadcast Desk Updates are teacher-created production posts that students can open, follow, and submit work into. Keep the update practical for a high school production class and use placeholders for missing source material.
```
