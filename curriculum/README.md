# Curriculum Source of Truth

This folder is the canonical curriculum and content source for DCC Creative Studio.

It contains:

- source transcripts and notes
- program area definitions
- human-readable lesson maps
- website-ready seed data
- reusable ChatGPT Pro prompts
- matching Drive folder structure documentation

## Core Rule

The website should be a curriculum and media project player. Lesson pages, video production projects, broadcast updates, and submission workflows should eventually render from structured data, not hardcoded one-off pages.
