# Curriculum Source of Truth

This folder is the canonical curriculum and content source for DCC Creative Studio.

It contains:

- source transcripts and notes
- program area definitions
- human-readable lesson maps
- website-ready seed data
- reusable ChatGPT Pro prompts
- matching Drive folder structure documentation

## Core Rule

The website should be a curriculum and media project player. Lesson pages, video production projects, broadcast updates, and submission workflows should eventually render from structured data, not hardcoded one-off pages.

## Phase 9 Curriculum Handoff

Phase 9 treats this folder as the source of truth for curriculum completion and slide handoff.

- Q3 Unreal Castle Documentary lesson folders live in `curriculum/pilot-batch/unreal/q3/`.
- Q2 Video Production Studio and DaVinci Resolve lesson folders live in `curriculum/pilot-batch/video-production/q2/`.
- Website-ready seed data lives in `curriculum/website-data/`.
- The ChatGPT Pro slide handoff package lives in `curriculum/slide-handoff/`.

Missing slide decks should be created outside this repository from the handoff briefs. Do not add PowerPoint files, generated slide images, fake Drive URLs, credentials, rosters, private student data, or raw media uploads to this folder.
