# Slide Link Return Template

After ChatGPT Pro creates decks and the teacher uploads them to Google Drive, fill in this template and return it to Codex so slide URLs and statuses can be updated accurately.

| lessonId | deck filename | Google Drive URL | status | date uploaded |
| --- | --- | --- | --- | --- |
|  |  |  | created | YYYY-MM-DD |

## Status Values

- `created`: the deck exists and the Google Drive URL is real.
- `needs-revision`: the deck exists but should not be linked yet.
- `not-created`: the deck was not generated.

Do not add placeholder or fake URLs.
